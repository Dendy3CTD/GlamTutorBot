# Если при запуске выводится только "Python"

На Windows команда `python` часто открывает **заглушку Microsoft Store**, а не установленный Python. Тогда скрипт не запускается.

## Что сделать

### Вариант 1: Использовать лаунчер `py`
В том же каталоге в командной строке выполните:
```bash
py -3 main.py
```
или просто:
```bash
py main.py
```
Если Python установлен с python.org, обычно срабатывает именно так.

### Вариант 2: Установить Python заново
1. Скачайте установщик: https://www.python.org/downloads/
2. Запустите установщик.
3. **Обязательно отметьте галочку "Add Python to PATH"** внизу окна.
4. Нажмите "Install Now".
5. Закройте и снова откройте командную строку / PowerShell.
6. Проверьте: `py -3 --version` или `python --version`.
7. Запустите бота: `py -3 main.py` или `python main.py`.

### Вариант 3: Запуск через PowerShell-скрипт
В папке проекта выполните:
```powershell
powershell -ExecutionPolicy Bypass -File run.ps1
```
Скрипт сам попробует найти рабочий Python (`py`, `python3`, `python`).

---

После успешного запуска в консоли должно появиться сообщение вроде:
```text
Бот запущен: @ваш_бот (id=...)
```
Окно не закрывайте — бот работает, пока оно открыто.
