# Инструкция по запуску бота

## Шаг 1: Установка Python

Если Python не установлен или не работает:

1. Скачайте Python с официального сайта: https://www.python.org/downloads/
2. При установке обязательно отметьте галочку "Add Python to PATH"
3. Установите Python

## Шаг 2: Установка библиотеки pyTelegramBotAPI

Откройте командную строку (PowerShell или CMD) в папке проекта и выполните:

```bash
pip install pyTelegramBotAPI
```

Или используйте файл requirements.txt:

```bash
pip install -r requirements.txt
```

## Шаг 3: Запуск бота

### Вариант 1: Через командную строку
```bash
python main.py
```

### Вариант 2: Через batch файл
Дважды кликните на файл `start_bot.bat`

### Вариант 3: Через PowerShell
```powershell
cd C:\Users\xiaomi\code\portfolio\GlamTutorBot
python main.py
```

## Шаг 4: Проверка работы бота

1. Откройте Telegram
2. Найдите вашего бота по имени или username
3. Отправьте команду `/start`
4. Проверьте работу всех функций:
   - Каталог товаров
   - Цены
   - Контакты
   - Адрес
   - Заказ товаров

## Устранение проблем

### Если Python не найден:
- Убедитесь, что Python установлен и добавлен в PATH
- Попробуйте использовать `py` вместо `python`
- Попробуйте использовать полный путь к Python

### Если библиотека не установлена:
```bash
python -m pip install --upgrade pip
pip install pyTelegramBotAPI
```

### Если бот не отвечает:
- Проверьте, что токен бота правильный в файле main.py
- Убедитесь, что бот запущен (окно командной строки должно быть открыто)
- Проверьте интернет-соединение

## Остановка бота

Для остановки бота нажмите `Ctrl+C` в окне командной строки, где он запущен.
