# Установка Python на Windows

У вас не установлен Python (команды `py` и `python` не работают). Сделайте так:

## Шаг 1: Скачать Python

1. Откройте в браузере: **https://www.python.org/downloads/**
2. Нажмите большую жёлтую кнопку **"Download Python 3.x.x"**.

## Шаг 2: Установить

1. Запустите скачанный файл (например `python-3.12.x-amd64.exe`).
2. **Важно:** внизу первого окна включите галочку:
   - **[x] Add python.exe to PATH**
3. Нажмите **"Install Now"**.
4. Дождитесь окончания установки, нажмите **"Close"**.

## Шаг 3: Перезапустить терминал

Закройте PowerShell или CMD и откройте его снова (или откройте новое окно).

## Шаг 4: Запустить бота

В папке проекта выполните:

```bash
python main.py
```

Должно появиться сообщение: `Бот запущен: @ваш_бот`. Окно не закрывайте — бот работает.

---

Если снова выводится только "Python" — при установке не была отмечена галочка "Add python.exe to PATH". Переустановите Python и включите её.
